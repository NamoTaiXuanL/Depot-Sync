# 代码库扫描工具 v0.2

## 项目概述
代码库扫描工具是一个智能文件同步系统，用于扫描和同步代码库文件。支持全盘扫描和指定文件夹扫描，提供增量同步功能。

## 功能特性
- 全盘扫描或指定文件夹扫描
- 智能文件哈希比较（基于文件大小和修改时间）
- 增量同步（只同步变化的文件）
- 图形用户界面（基于Tkinter）
- 多线程同步处理
- 权限错误自动处理
- 驱动器UUID识别

## 使用说明

### 扫描选项
1. **全盘扫描**: 扫描所有硬盘驱动器中的代码库
2. **选择特定文件夹**: 手动选择要扫描的文件夹

### 同步选项
1. 选择同步目标路径
2. 开始同步（支持增量同步）

### 界面说明
- **源代码库信息**: 显示扫描到的代码库详细信息
- **同步代码库信息**: 显示同步状态和结果
- **同步记录**: 显示同步操作的历史记录

## 技术实现
- 使用文件哈希比较实现增量同步
- 多线程处理提高扫描效率
- PowerShell获取硬盘UUID信息
- JSON配置文件保存同步历史
